Kompressor

Compress PHP code into smaller file size

Kompressor is a Java program which compresses PHP files via removing unnecessary white-space from them.

Benefits

Smaller files mean faster deployment of PHP applications.

NOTE: It's not proven that compressed files run any faster than uncompressed. Except for the fact that smaller files are fetched faster from the background storage.

Line-breaks are not removed. The reason behind this is that debugging the scripts without line numbers is impossible. It would make program-parts unrecognizable. Removing other kinds of white-space does not have such a dramatic effect on readability.

How to use?

You should wire-in this program into your build script. It can receive a directory of input files via program arguments and it will compress the files it founds. The output folder can also be specified.

Legal

Certain PHP-libs have their own license which prohibits you from modifying them. The output of this program is equal to the input in execution terms but results in a program-code which has all it's comments (including licenses) removed.

Thus the output is legal to deploy on a server but might be illegal to publish in any other form.
